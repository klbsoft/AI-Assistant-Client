## ðŸš€ Getting Started

### Prerequisites

- Rust (latest stable)  
- npm or yarn  

### Installation

1. **Clone the repository**
```bash
git clone https://github.com/klbsoft/AI-Assistant-Client.git
cd AI-Assistant-Client
```

2. **Install frontend dependencies**
```bash
cd AI-Assistant-Client
npm install
```

3. **Build the backend**
```bash
cd api
cargo build
```


4. **Run the development servers**
```bash
npm run dev
```
